skills

HTML is the standard markup language for creating the structure and content of web pages, including emails. In email development, HTML is used to define the structure of the email, including text, images, links, and other elements.

CSS is used for styling and formatting the content defined by HTML. In email development, inline styles are often used to ensure compatibility across various email clients. CSS is crucial for achieving a consistent and visually appealing layout in emails.

JavaScript used to enhance interactivity within an email, such as implementing image carousels or dynamic content. However, its usage is limited, and caution must be taken to ensure compatibility.

Figma is a design tool that allows designers to create and collaborate on designs. In email development, Figma is often used to create the visual mockups of email campaigns.

Photoshop is an image editing software. In email development, designers might use Photoshop to edit and optimize images for use in email campaigns.

Similar to Figma, it is used to create visual designs and layouts for email campaigns.

In email development, Litmus is used to test how email campaigns will appear and function across various email clients and devices.
